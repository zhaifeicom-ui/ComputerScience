# 用户操作指南

本指南面向最终用户与系统管理员，覆盖常用工作流与操作要点。

登录与导航
- 通过前端登录页面进行身份认证，成功后进入数据大盘、商品管理、库存管理等模块。

库存管理工作流
- 创建库存记录
  - 路径：Inventory -> 新增库存
  - 必填：productId、stock、safetyStock
- 查看库存明细
- 更新库存
- 删除库存
- 库存预测检查
  - 指定 futureDays，查看预计缺货风险和需要关注的产品列表
- 通过销售数据更新库存（扣减/恢复）

商品管理
- 查看/创建/修改/删除商品信息
- 商品与库存的映射关系在库存记录中体现

导入任务（Jobs）
- 运行导入任务以将外部数据导入系统
- 查看任务状态、日志与结果

数据可视化与报表
- 数据大盘提供关键指标的可视化呈现，帮助快速了解库存与销量趋势

解读与告警
- 预测告警：当预测销量超过可用库存时，系统将给出警告
- 理解各字段：库存、安全库存、可用库存、预测销量、未来天数等

故障排查与支持
- 如遇认证问题、接口异常、数据缺失，请联系开发/运维团队提供日志与环境信息。

附注
- 本指南与 docs/API.md 共同构成对系统功能的日常使用手册。具体实现可能随后端改动而变动，请以最新接口文档为准。
